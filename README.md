pubs_manager
=======

Scripts to extract and annotate pubmed articles
  
## Installation
  ```
  pip install -r requirements.txt
  ```
## Usage
  
  Extract XML articles from PMID list
  ```
  python get_pubs.py
  ```
  Call service to annotate XML articles
  ```
  python get_ann.py
  ```
